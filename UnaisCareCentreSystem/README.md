# Inventory Management System

## Overview
This project is an Inventory Management System designed to help users manage products, stock, orders, and reports efficiently. The application consists of several HTML files, each serving a specific purpose within the system.

## Project Structure
```
inventory-management-system
├── src
│   ├── HomePage.html
│   ├── Schedules.html
│   ├── Inventory.html
│   ├── Requests.html
│   ├── Billing.html
│   ├── Report.html
│   ├── Settings.html
│   ├── Login.html
│   ├── css
│   │   └── styles.css
│   └── js
│       └── main.js
└── README.md
```

## Files Description

- **src/HomePage.html**: The main entry point of the inventory management system. It contains a header and a dashboard with links to manage products, stock, orders, and reports.

- **src/products.html**: This file is used to manage products. It includes a header and a section to display product details, forms to add or edit products, and buttons for actions like saving or deleting products.

- **src/stock.html**: Displays stock information, including items in stock and related actions.

- **src/orders.html**: Manages orders, showing pending orders and options to process them.

- **src/reports.html**: Allows users to generate and view reports related to inventory and sales.

- **src/css/styles.css**: Contains the styles for the HTML files, ensuring a consistent look and feel across the application.

- **src/js/main.js**: Includes JavaScript functionality for the application, such as handling form submissions, dynamic content updates, and any interactive features.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `src` folder and open `HomePage.html` in a web browser to view the application.
3. Navigate through the links to manage products, stock, orders, and reports.

## Features
- User-friendly interface for managing inventory.
- Ability to add, edit, and delete products.
- View stock levels and manage orders.
- Generate and view reports for better decision-making.

## Usage Guidelines
- Use the dashboard on the homepage to navigate to different sections of the inventory management system.
- Follow the prompts in each section to manage inventory effectively.
